* `+setplaylistvaroverride`

* `ns_private_match_only_host_can_change_settings`

| Value | Description                              |
|-------|------------------------------------------|
| `0`   | Players can change all match settings    |
| `1`   | Players can only change map and gamemode |
| `2`   | Players can change nothing               |
